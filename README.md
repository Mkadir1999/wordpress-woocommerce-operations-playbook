# WordPress / WooCommerce Operations Playbook

A practical operations checklist for running and troubleshooting a service-based WordPress + WooCommerce website.

## What this proves

- WordPress administration
- WooCommerce operations
- SEO structure
- Plugin troubleshooting
- Web business operations
- Customer conversion thinking

## Daily Checklist

- [x] Website loads correctly
- [x] Checkout works
- [x] Contact form works
- [x] SSL is valid
- [x] Important pages are indexed
- [x] No critical plugin errors
- [x] Backups are running

## SEO Page Checklist

- One clear H1
- SEO title under 60 characters
- Meta description under 160 characters
- Internal links added
- Service area included naturally
- Clear call to action
- FAQ section added if useful
- Schema enabled if appropriate

## WooCommerce Checkout Test

1. Add product/service to cart
2. Confirm price
3. Apply coupon if needed
4. Test payment method
5. Confirm order email
6. Confirm admin notification
7. Confirm thank-you page

## Plugin Conflict Process

1. Backup site
2. Disable cache/minify
3. Disable recently added plugins
4. Switch theme only if necessary
5. Check browser console
6. Check PHP error logs
7. Reactivate one plugin at a time
